# 🌠 접근방식
<img width="100%" alt="접근 방식" src="https://github.com/M-05/Mountain_Ghost/assets/103846429/2c91e5eb-1b66-47b2-be3f-0dda26fc6672">

- 이미 관련 아이디어 및 정책들이 무궁무진하다.  
- 이미 관련 컨텐츠도 많지만 타켓층이 20대가 아니다.  
- 홍보가 잘 안되있고 안이쁘게 꾸몄다.
- 이미 관련 어플들도 존재하지만 접근의 어려움과 지속가능하지 않다.
  
<img width="40%" alt="접근 방식" src="https://github.com/M-05/Mountain_Ghost/assets/103846429/288a0cc6-4bad-4b69-b8b2-20ce0baaee59"> <img width="50%" alt="접근 방식" src="https://github.com/M-05/Mountain_Ghost/assets/103846429/c69ee319-8153-4019-aa50-771921da8e0c">

1. 주로 도시 환경에 거주하며 모바일, 디지털 문화에 친화적인 20대를 주 사용자로 설정한다.  
2. 핫 플레이스 + 지하철 내부에 산림 포토존을 이쁘게 꾸민다.(SNS 홍보)  
3. “MG” 어플 : 생성형 AR 캐릭터  
→ 산림에 관심 없는 20대를 산림으로 가게 유도하여 나도 모르는 사이에 즐거움, 힐링, 건강, 치유를 1차 목표를 합니다.  
→ 2차 목표는 대기업이 스스로 자발적으로 움직이길 바랍니다.
